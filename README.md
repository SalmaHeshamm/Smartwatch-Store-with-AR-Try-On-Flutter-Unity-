# A Flutter–Unity Based Smartwatch Store with AR Try-On

This project presents a hybrid Flutter–Unity application that functions as a complete smartwatch store while providing an Augmented Reality (AR) try-on experience. The system enables users to browse smartwatch models, view product details, and visualize watches directly on their wrist using AR technology powered by Unity and AR Foundation.

---

## 🔥 Key Features

### 🛍️ Smartwatch Store
- Browse a catalog of smartwatch models  
- View product images, specifications, and pricing  
- Clean and modern Flutter UI  
- Favorites and cart system 

### 📱 AR Try-On Module
- Real-time smartwatch visualization on the user's wrist  
- Built using Unity + AR Foundation  
- Smooth Flutter–Unity communication  
- Supports different watch models and textures

### 🧩 Tech Stack
- **Flutter** – Mobile UI & app logic  
- **Unity (AR Foundation)** – AR rendering & tracking  
- **flutter_unity_widget_2** – Communication layer  
- Languages: **Dart & C#**  
- Platforms: **IOS and Android**

## 🚀 Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/SalmaHeshamm/A-Flutter-Unity-Based-Smartwatch-Store-with-AR-Try-On.git
cd A-Flutter-Unity-Based-Smartwatch-Store-with-AR-Try-On
```

### 2. Install Flutter Dependencies
```bash
flutter pub get
```

### 3. Run the Application
```bash
flutter run
```
--- 

## Flutter–Unity Communication
The project uses flutter_unity_widget_2, ensure Unity and Flutter versions are compatible with it.
